## Core ideas
LangChain is an open-source framework that provides a pre-built agent architecture and integrations for building AI agents. Its design allows for flexibility and adaptability, enabling developers to build agents that can evolve quickly with the ecosystem. The framework supports a wide range of models and tools, making it a versatile choice for developers. For example, LangChain's `create_agent` function provides a proven ReAct pattern on LangGraph's durable runtime, allowing developers to build agents in minutes [1]. This functionality enables developers to focus on building applications rather than rebuilding the underlying infrastructure.
