# Agentic coding

## Why it's on the radar today
Agentic coding has surfaced in the developer world today due to the emergence of tools like Claude Code, an agentic coding tool that lives in your terminal and helps with routine tasks, explaining complex code, and handling Git workflows [1]. This tool has been featured on GitHub Trending, HN Front Page, Lobste.rs, and dev.to, indicating a growing interest in agentic coding. Additionally, the launch of hosted MCP servers has connected tools like Claude, Cursor, and Grok, changing the landscape of development [2].

## What's being discussed
The current conversation around agentic coding involves specific tools and techniques, such as agent skills, which are reusable plugins for AI coding assistants [4]. These skills are structured instructions that teach AI agents how to handle specific tasks, and they are not just prompts, but rather micro-APIs for agent behavior. The concept of loop engineering is also being discussed, which involves the planning, writing, checking, and fixing of code by AI agents [3]. Furthermore, the rise of agent harnesses is enabling developers to use AI coding agents more effectively, by providing a way to manage and improve the loops that these agents use [4]. Some notable projects and tools include Claude Code, Cursor, and Aider, which are being used by companies like We The Flywheel to build AI-native workflows [5].

## Core ideas
Agentic coding involves using AI agents to automate routine tasks and improve development workflows. These agents use loops to plan, write, check, and fix code, and they can be taught new tasks using agent skills. For example, Claude Code can be used to automate Git workflows, such as creating and merging pull requests, by using natural language commands [1]. Agent skills can be used to teach Claude Code how to handle specific tasks, such as code review or testing. Loop engineering involves managing and improving these loops to make the agents more effective.

## Where it fits
Agentic coding is a key part of the broader trend towards AI-native development, where AI is used to automate and improve development workflows. It fits into the stack by providing a way to automate routine tasks, improve code quality, and reduce the workload of human developers. Developers can use agentic coding tools and techniques to improve their workflows and make themselves more productive.

## Do this in 5 minutes
To get started with agentic coding, you can try using Claude Code to automate a simple Git workflow. For example, you can use the following command to create a new branch and commit a change:
```bash
claude code create-branch my-new-branch
claude code commit -m "My new commit"
```
This will create a new branch and commit a change using Claude Code. You can then use the `claude code` command to automate other Git workflows, such as merging pull requests or creating new repositories.
