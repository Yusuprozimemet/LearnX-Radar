## Core ideas
Claude is a large language model developed by Anthropic, with a range of applications including coding, chatbots, and generative pre-trained transformers [1]. At its core, Claude is designed to provide a foundation for various tasks, from simple text generation to complex automation workflows. One of the key concepts in Claude is the idea of Skills, which are essentially pre-built functions that can be used to perform specific tasks [2]. These Skills can be combined and customized to create more complex workflows, making Claude a powerful tool for developers. For example, Claude Code allows developers to use Skills to automate coding tasks, such as generating boilerplate code or debugging [3].

## Do this in 5 minutes
To get hands-on with Claude, try redirecting Claude Code requests to a custom backend using the `ANTHROPIC_BASE_URL` environment variable. Run the following command in your terminal:
```bash
export ANTHROPIC_BASE_URL=https://your-proxy.example.com
export ANTHROPIC_API_KEY=your-key
claude
```
Replace `https://your-proxy.example.com` with your custom backend URL and `your-key` with your actual API key. This will allow you to test and experiment with Claude's capabilities in a custom environment [3].
