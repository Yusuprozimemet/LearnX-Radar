# Agentic coding
## Why it's on the radar today
Agentic coding has recently gained attention in the developer world due to its increasing presence in various repositories and job postings, such as those from Akkio and Grafana Labs, which are leveraging agentic workflows and AI-native applications for large media agencies and observability tasks, respectively [1, 5].

## What's being discussed
The current conversation around agentic coding revolves around its application in building AI-native applications, such as those used by Akkio for audience building and distribution, measurement, and campaign analysis [1]. Additionally, Grafana Labs is experimenting with autonomous SREs, an agentic platform, and improving agentic dev experience with o11y insights [5]. ## Core ideas
Agentic coding involves the use of autonomous agents to perform tasks that would typically require human intervention. These agents can be used to automate various aspects of software development, deployment, and maintenance. For instance, Akkio is using agentic workflows to build AI-native applications for large media agencies, which involves tasks such as audience building and distribution, measurement, and campaign analysis [1]. In another example, Grafana Labs is experimenting with autonomous SREs, an agentic platform, and improving agentic dev experience with o11y insights [5].

## Where it fits
Agentic coding fits into the broader stack of software development, particularly in the context of AI-native applications and observability tasks. It can be used to automate various aspects of software development, deployment, and maintenance, making it an attractive option for companies looking to improve their development workflows.
