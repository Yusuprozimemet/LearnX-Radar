## Core ideas
At its core, an AI agent is a program that perceives its environment and takes actions to achieve a specific goal or set of goals. AI agents can be classified into different types based on their characteristics and capabilities. For example, a simple reflex agent reacts to the current state of the environment without considering future consequences, whereas a model-based reflex agent uses a model of the environment to make decisions. A goal-based agent, on the other hand, uses a goal or set of goals to guide its decision-making process. To illustrate this concept, consider a simple thermostat that adjusts the temperature based on the current reading - this can be seen as a basic example of a simple reflex agent.

## Do this in 5 minutes
To get hands-on experience with AI agents, open a Python interpreter and run the following code:
```python
import random

class SimpleReflexAgent:
    def __init__(self):
        self.temperature = 20

    def perceive(self):
        self.temperature += random.uniform(-1, 1)

    def act(self):
        if self.temperature < 18:
            print("Heating")
        elif self.temperature > 22:
            print("Cooling")
        else:
            print("Maintaining temperature")

agent = SimpleReflexAgent()
for _ in range(10):
    agent.perceive()
    agent.act()
```
This code defines a simple reflex agent that adjusts the temperature based on the current reading, illustrating the basic concept of an AI agent. Run this code to see the agent in action and experiment with different scenarios to understand how AI agents can be used in various applications.
