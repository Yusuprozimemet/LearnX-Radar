# Coding agents

## Why it's on the radar today
Coding agents have surfaced in the developer world today due to their use in Conductor, a platform that enables builders to run teams of coding agents [1]. This concept has also been discussed on GitHub Trending, HN Hiring, and dev.to, indicating a growing interest in this technology. Conductor's recent $22m Series A announcement has further highlighted the potential of coding agents [1].

## What's being discussed
The current conversation around coding agents involves specific tools and techniques, such as the AI coding agent harness, which is a crucial component in determining the real-world performance of AI coding agents [2]. Developers are also exploring different coding agents like Claude Sonnet 5, Fable 5, and GitHub Copilot, and evaluating their strengths and weaknesses [3]. Additionally, projects like agents-cli and agentsview are providing developers with the skills and tools to build, deploy, and manage coding agents on platforms like Google Cloud [4, 5].

## What you'll learn
* How to build and deploy coding agents using tools like agents-cli
* The importance of the AI coding agent harness in determining the performance of coding agents
* How to evaluate and compare different coding agents like Claude Sonnet 5 and Fable 5
* How to use agentsview to manage and track coding agents
* How to integrate coding agents with existing development workflows

## Core ideas
The core idea behind coding agents is to enable developers to automate repetitive tasks and focus on more complex and creative work. The AI coding agent harness is a critical component in this process, as it determines how the coding agent interacts with the development environment and performs tasks [2]. For example, a coding agent like Claude Sonnet 5 can be used to automate tasks like code completion and bug fixing, but its performance is highly dependent on the quality of the harness [3]. A well-designed harness can enable the coding agent to work seamlessly with the development environment, while a poorly designed harness can lead to errors and inefficiencies.

## Where it fits
Coding agents are part of a broader trend towards automation and AI-powered development tools. They can be used to augment existing development workflows, automate repetitive tasks, and improve overall productivity. Developers can use coding agents to automate tasks like code review, testing, and deployment, and focus on more complex and creative work.

## Do this in 5 minutes
To get started with coding agents, you can install agents-cli using the following command:
```bash
uvx google-agents-cli setup
```
This will give you the skills and commands to build, deploy, and manage coding agents on Google Cloud. You can then use the `agents-cli` command to build your first coding agent and start exploring the possibilities of coding agents.
