# Agentic coding
## Why it's on the radar today
Agentic coding has surfaced in the developer world today with the release of OpenMontage, the world's first open-source, agentic video production system [1]. This system has gained attention on GitHub Trending and has sparked interest in the development community.

## What's being discussed
The current conversation around agentic coding revolves around the use of agents in video production, with OpenMontage being a prime example [1]. The system utilizes 12 pipelines, 52 tools, and 500+ agent skills to turn an AI coding assistant into a full video production studio. Additionally, the A2A Protocol is being discussed as a foundation for building multi-agent systems [3]. There is also a focus on the security aspects of agentic coding, with discussions around preventing prompt injection in LangChain Python apps [5].

## What you'll learn
* How to use agents in video production with OpenMontage
* The basics of the A2A Protocol for building multi-agent systems
* How to prevent prompt injection in LangChain Python apps
* How agentic coding fits into the broader stack
* A hands-on exercise to get started with agentic coding

## Core ideas
Agentic coding involves the use of agents to automate tasks and make decisions. In the context of OpenMontage, agents are used to automate video production tasks such as editing and color correction. The A2A Protocol provides a foundation for building multi-agent systems, allowing agents to communicate and work together seamlessly. However, with the use of agents comes the risk of prompt injection, which can be prevented by using secure coding practices.

## Where it fits
Agentic coding fits into the broader stack as a way to automate tasks and make decisions. It can be used in a variety of applications, from video production to document processing. By using agents, developers can create more efficient and effective systems.

## Do this in 5 minutes
Try out OpenMontage by cloning the repository and running the quick start guide [1]. This will give you a hands-on introduction to agentic coding and the use of agents in video production.
```bash
git clone https://github.com/calesthio/OpenMontage.git
cd OpenMontage
python quick_start.py
```
