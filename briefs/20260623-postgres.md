## Core ideas
Postgres extensions like PlanetScale offer a way to improve the performance and scalability of Postgres databases. These extensions can provide features such as horizontal partitioning, which allows for more efficient data storage and retrieval. To use Postgres extensions, developers need to understand how to configure and manage them, as well as how to integrate them with other technologies.

For example, to use PlanetScale with Next.js and TypeScript, a developer might create a Postgres database with a PlanetScale extension, and then use a library like `pg` to connect to the database from their Next.js application. They would also need to configure the PlanetScale extension to optimize performance and scalability.

## Do this in 5 minutes
Create a new Postgres database with a PlanetScale extension using the following command:
```bash
pscale create db mydb --region us-east-1 --nodes 2
```
This command creates a new Postgres database with a PlanetScale extension, with two nodes in the us-east-1 region. You can then connect to the database using a library like `pg` to start exploring the features of Postgres and PlanetScale.
