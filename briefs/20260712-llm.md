## Core ideas
To build real AI applications, an LLM needs to be connected to external tools and APIs. This can be achieved through a process called tool calling or function calling. For example, if a user asks a chatbot to "Book the cheapest flight from Mumbai to Tokyo", the LLM can call a flight booking API to execute the action [1]. This process involves the LLM understanding the request, determining the required tool or API, calling the API, receiving the results, and generating a grounded response.

## Where it fits
LLM is a key technology in the broader stack of AI and machine learning. It is used in building chatbots, virtual assistants, and other AI-powered applications. Developers can use LLM in conjunction with other technologies, such as React and Node.js, to build AI-native applications.

## Do this in 5 minutes
Try calling a simple API using a Python script to understand the concept of tool calling. For example, you can use the `requests` library to call the OpenWeatherMap API to get the current weather for a specific location. Here is a simple code snippet to get you started:
```python
import requests

api_key = "YOUR_API_KEY"
location = "London"

response = requests.get(f"http://api.openweathermap.org/data/2.5/weather?q={location}&appid={api_key}")

print(response.json())
```
Replace `YOUR_API_KEY` with your actual API key and run the script to see the response from the API.
